# React UI Kit

一套基于 React 的通用 UI 组件库，包含常用的基础组件。

## 特性

- 🎨 现代化设计风格
- 📦 开箱即用的组件
- 🎯 TypeScript 支持（待添加）
- 📱 响应式设计
- 🎨 可定制主题

## 安装

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 组件列表

- Button - 按钮组件
- Card - 卡片组件  
- Input - 输入框组件

## 使用示例

```jsx
import { Button, Card, Input } from 'react-ui-kit'

function App() {
  return (
    <div>
      <Button variant="primary">主要按钮</Button>
      <Card title="标题">内容</Card>
      <Input placeholder="请输入" />
    </div>
  )
}
```
